# React + Vite
Образовательная платформа
О проекте
Я разработал полноценную образовательную платформу (LMS) на React, которая позволяет создавать курсы, управлять уроками, проводить тестирование и отслеживать прогресс студентов. Это мой личный проект, в котором я реализовал все необходимые функции для онлайн-обучения.

Технологии
React 18 — библиотека для пользовательских интерфейсов

React Router DOM — маршрутизация

LocalStorage — хранение данных

CSS Modules — стилизация компонентов

 Возможности
Для всех пользователей:
Регистрация и авторизация

Просмотр списка курсов

Изучение уроков

Прохождение тестов

Смена темы (светлая/темная)

Для студентов:
Добавление курсов в личный кабинет

Отслеживание прогресса обучения

Завершение курсов с получением сертификата

Просмотр результатов тестов

Для преподавателей (админов):
Создание, редактирование и удаление курсов

Добавление уроков в курсы

Создание тестов с вопросами

Просмотр результатов студентов в деталях курса

Управление всем контентом платформы

 Установка и запуск
Требования:
*Node.js* (версия 14 или выше)
*npm или yarn*

Шаги для установки:
bash
# Клонирую репозиторий
https://github.com/LittleRaccoonn/Students-Platform
# Перехожу в папку проекта
cd learning-platform

# Устанавливаю зависимости
npm install

# Запускаю проект
npm run dev
После запуска проект будет доступен по адресу: http://localhost:5173

Основные действия:
Регистрация — создайте новый аккаунт

Добавление курсов — на странице Dashboard выберите интересующие курсы

Обучение — переходите к урокам и изучайте материалы

Тестирование — после уроков проходите тесты для проверки знаний

Завершение курса — после всех уроков отметьте курс как завершённый

Для администратора:
Создание курса — в админ-панели создайте новый курс

Добавление уроков — внутри курса добавьте уроки с контентом

Создание тестов — к каждому уроку создайте тест с вопросами

Аналитика — в деталях курса смотрите результаты студентов

 Особенности дизайна
Адаптивный дизайн — корректно отображается на всех устройствах (кроме телевоня)

Тёмная тема — поддержка светлой и тёмной темы оформления

Современные карточки — с тенями и анимациями при наведении

Прогресс-бары — визуальное отображение результатов тестов

Безопасность
Защищённые маршруты для авторизованных пользователей

Разграничение прав доступа (студент / администратор(учитель))

Проверка ролей при доступе к админ-функциям

 
 Известные проблемы
Нет ограничения на количество вопросов в тесте

Изображения профиля сохраняются в Base64 (большой объём данных)

Все данные хранятся в LocalStorage (очищаются при очистке кэша браузера)
